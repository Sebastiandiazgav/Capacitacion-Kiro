# Instalación de Kiro

## Requisitos del sistema

Kiro está disponible para macOS, Windows y Linux.

- **macOS:** Intel y Apple Silicon con las últimas actualizaciones de seguridad.
- **Windows:** Windows 10 y 11 (solo 64-bit). ARM no soportado actualmente.
- **Linux:** glibc 2.39 o superior (Ubuntu 24+, Debian 13+, Fedora 40+, Arch Linux, Linux Mint 22+).

## Pasos de instalación

### 1. Descargar Kiro

1. Ir a [kiro.dev](https://kiro.dev) y descargar el instalador para tu sistema operativo.

### 2. Instalar

1. Abrir el archivo descargado.
2. Seguir las instrucciones de instalación según tu SO.
3. Abrir Kiro IDE.

### 3. Primer inicio

1. Kiro te pedirá iniciar sesión con el proveedor de tu elección (social login o AWS, correo). Más info sobre [métodos de autenticación](https://kiro.dev/docs/getting-started/authentication.md).

2. Puedes [importar tu configuración de VS Code](https://kiro.dev/docs/guides/migrating-from-vscode.md) (extensiones y settings). Si usas otro editor, puedes saltar este paso. Selecciona tu tema preferido y permite que Kiro configure la integración de shell para ejecutar comandos.

3. Llegarás a la página de bienvenida. Abre un proyecto para comenzar. Más info en [tu primer proyecto](https://kiro.dev/docs/getting-started/first-project.md).

## Downgrade a una versión anterior

Si una actualización causa problemas, puedes volver a una versión anterior:

1. Ir a la [página de descargas](https://kiro.dev/downloads).
2. Bajar hasta la lista de versiones.
3. Expandir la versión deseada (ejemplo: "IDE 0.12.x").
4. Descargar el instalador para tu plataforma.
5. Desinstalar la versión actual:
   - **macOS:** arrastrar Kiro de Aplicaciones a la Papelera.
   - **Windows:** desinstalar desde **Configuración > Aplicaciones > Aplicaciones instaladas**.
   - **Linux:** `sudo apt remove kiro` o `sudo dnf remove kiro`.
6. Ejecutar el instalador descargado.

> Tus settings, extensiones y sesión se conservan entre reinstalaciones.
